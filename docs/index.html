Implementation & Lessons
구현 방법 & 시행착오
🛠 구현 방법
Firestore 실시간 동기화
Stream + Provider로 캡슐 목록을 실시간 반영. 캡슐 저장 즉시 지도 마커와 목록 자동 업데이트
Geofencing 구현
Geolocator 위치 스트림 + 거리 계산으로 100m 이내 진입 감지 → 로컬 알림 발송
커스텀 마커
dart:ui Canvas로 북마크 아이콘을 BitmapDescriptor로 변환하여 지도에 적용
⚠️ 주요 시행착오
한글 경로 문제
사용자명 '징규'가 Android 빌드 경로에 포함되어 오류 → 프로젝트를 영어 경로로 이동
Firestore 복합 인덱스
where + orderBy 복합 쿼리는 인덱스 필수 → Firebase Console에서 수동 생성
에뮬레이터 RAM 부족
기본 2GB 설정에서 앱 크래시 → 4GB로 증설 후 안정화
