장소와의 단절

✦ "그 장소에 다시 갔을 때, 과거의 내가 남긴 편지를 받는 경험"

Vision & Solution
장소가 곧 일기장이 되는 세상
📍
위치 기반 캡슐
특정 좌표에 사진 + 메모를 캡슐로 저장. GPS로 현재 위치를 자동 기록합니다.
🌍
공개 or 비공개 피드
다른 사람이 같은 장소에 남긴 공개 기억을 탐색하며 공감이 가능합니다.
⏳
장소가 기억을 깨운다
걸어다니면서 과거의 나를 만나는 경험 — 같은 장소를 다시 방문하면 그때의 기억이 되살아납니다. 일상의 모든 장소가 개인의 타임캡슐이 됩니다.
