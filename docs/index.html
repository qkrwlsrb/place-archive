Implementation & Lessons
구현 방법 & 시행착오
🛠 구현 방법
Firestore 실시간 동기화
Stream + Provider로 캡슐 목록을 실시간 반영. 캡슐 저장 즉시 지도 마커와 목록 자동 업데이트
Geofencing 구현
Geolocator 위치 스트림 + 거리 계산으로 100m 이내 진입 감지 → 로컬 알림 발송
AuthGate 자동 라우팅
Firebase authStateChanges 스트림을 AuthGate가 구독. 로그인 상태 변화 시 화면 자동 전환
⚠️ 주요 시행착오
한글 경로 빌드 오류
사용자명이 포함된 Android 빌드 경로에서 오류 발생 → 프로젝트를 영어 경로로 이동하여 해결
Firestore 복합 인덱스
where + orderBy 복합 쿼리는 인덱스 필수 → Firebase Console에서 직접 생성하여 해결
Android v1 Embedding 오류
android 폴더 구조 누락으로 빌드 실패 → flutter create로 android 폴더 재생성 후 해결
에뮬레이터 RAM 부족
기본 2GB 설정에서 앱 크래시 반복 → AVD RAM을 4GB로 증설하여 안정화
