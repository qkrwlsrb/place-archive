# 요구사항 정의 (Requirements)

## MoSCoW 분류

### Must (반드시 있어야 함)

| # | 기능 | 설명 |
|---|------|------|
| M1 | 소셜 로그인 | Google / Apple 로그인 |
| M2 | 현재 위치 캡슐 생성 | GPS 좌표 + 사진 + 메모 저장 |
| M3 | 지도 위 캡슐 핀 표시 | 내 캡슐을 지도에서 시각화 |
| M4 | 재방문 감지 알림 | 저장 장소 반경 100m 진입 시 알림 |
| M5 | 공개/비공개 설정 | 캡슐별 공개 범위 선택 |

### Should (있으면 좋음)

| # | 기능 | 설명 |
|---|------|------|
| S1 | 타임라인 피드 | 내 캡슐을 날짜순으로 모아보기 |
| S2 | 타인 공개 캡슐 탐색 | 주변 공개 캡슐 지도에서 발견 |
| S3 | 음악 연결 | 현재 재생 곡을 캡슐에 첨부 |

### Could (여유 있으면 구현)

| # | 기능 | 설명 |
|---|------|------|
| C1 | 공감 / 댓글 | 공개 캡슐에 반응 남기기 |
| C2 | 캡슐 공유 링크 | 외부 공유 가능한 링크 생성 |
| C3 | 취향 태그 | 감정/날씨/장소 유형 태그 |

### Won't (이번 버전에서 제외)

| # | 기능 | 이유 |
|---|------|------|
| W1 | 실시간 채팅 | 범위 초과 |
| W2 | AR 뷰 | 기술 난이도 높음 |
| W3 | 유료 결제 | MVP 단계 불필요 |
