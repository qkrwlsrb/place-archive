# ADR-0002: 상태관리 방식 선택

- **날짜**: 2026-05-19
- **상태**: 확정

## 결정
**Provider** 패키지를 상태관리 솔루션으로 선택한다.

## 고려한 대안
| 대안 | 검토 결과 |
|------|-----------|
| Riverpod | 학습 곡선이 더 높음 |
| BLoC | 개인 프로젝트에 과도한 보일러플레이트 |
| GetX | 아키텍처 경계를 흐릴 수 있음 |

## 결정 이유
- Flutter 공식 권장 패키지
- ViewModel과 자연스럽게 연결됨
- 초급자도 개념 이해가 빠름

## 결과
- ChangeNotifier 상속한 ViewModel을 application/view_models/에 위치
