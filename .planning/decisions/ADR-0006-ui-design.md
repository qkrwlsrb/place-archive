# ADR-0006: UI 디자인 방향 선택

- **날짜**: 2026-05-26
- **상태**: 확정

## 결정
**감성 다이어리** 컨셉 — 베이지/크림 톤, 손글씨 폰트(Gaegu) 기반 UI를 적용한다.

## 고려한 대안
| 대안 | 검토 결과 |
|------|-----------|
| 클린 모던 (흰 배경, 볼드 타이포) | 일반적인 앱과 차별점 없음 |
| 다크 모드 (어두운 배경, 네온 포인트) | 타임캡슐 감성과 맞지 않음 |
| 감성 다이어리 (베이지/크림, 손글씨) | 앱 컨셉과 가장 잘 맞음 |

## 결정 이유
- 앱 핵심 컨셉(장소에 기억을 남기는 타임캡슐)과 일관성
- 20대 타겟 사용자의 감성 니즈와 부합
- Google Fonts의 Gaegu(한국어 손글씨 폰트)로 차별화된 브랜드 정체성 구현
- 베이지/크림 팔레트가 따뜻한 기억이라는 감성을 시각적으로 표현

## 결과
- Primary: `#B87A50` (따뜻한 테라코타)
- Background: `#FDF6EC` (크림)
- 제목/헤더: `GoogleFonts.gaegu()`
- 본문: `GoogleFonts.notoSans()`
- `google_fonts: ^6.2.1` 패키지 사용
